This repository (link) contains the implementation of the Explainable Artificial Intelligence (XAI) framework developed in the study [Quantifying atmospheric and land drivers of hot temperature extremes through explainable Artificial Intelligence](https://egusphere.copernicus.org/preprints/2025/egusphere-2025-5392/).  

### Scientific Objective

The DDM leverages a combination of land, ocean, sea ice, and atmospheric CO2 data as predictors to estimate two heatwave indices:
- number of extremely hot days
- a composite heatwave propensity index


### Methodology

The model architecture incorporates a Guided Hybrid Genetic Algorithm for feature selection and a Random Forest regressor for prediction.

To do the feature selection, a Guided Hybrid GEnetic Algorithm is used. This algorithm uses a genetic algorithm as the global search strategy, while doing local search operations to speed-up the convergence. The function the GHGA is wrapped around a a Random FOrest which minimizes a specific cost function. This Random Forest consists of 200 trees generated by bootstrapping the training dataset, and uses as a cost function the mean squarred error calculated by the samples ommited by the bootstrap sampling. GHGA was run sepparately for HWP and Tx90. 

At the end we selected the best models: that is, the ones whose predictors combination prvided the smallest MSE.


### Explainability (XAI)

For each ensemnle member , the importance od each predictor is quantified using SHAP values. 

### Scope

Forecasts are made for ten European regions across the main meteorological seasons, with the model trained on reanalysis datasets spanning 1950–2022. The ten regions have been grouped to account for their climate characteristics. Each region is tested independently from the others for the four main seasons: spring (MAM), summer (JJA), autumn (SON), and winter (DJF).
